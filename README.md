# Agent-Optima

Monorepo foundation for Phase 1 MVP.

## Workspace Layout

- apps/ — deployable applications (web, gateway, control API)
- packages/ — shared libraries (schemas, utils, SDKs)
- services/ — background workers
- infra/ — IaC and runtime deployment assets

## Getting Started

1. Install Node.js 20+
2. Run `npm install`
3. Run `npm run build`

## Current Scope

This commit bootstraps repository standards and shared event schemas.
